# Introduction

## Purpose
This tool is designed to analyze X-ray Photoelectron Spectroscopy (XPS) data acquired using **ULVAC-PHI's Phi series** instruments. It streamlines the workflow from raw data import to peak fitting.

## Requirements
To run XPS-analyzer, you need **Python 3.8+** installed on your system.
The following Python libraries are required:

* **numpy**
* **pandas**
* **scipy**
* **matplotlib**
* **openpyxl** (Required for exporting results to Excel)
